Fight4Life Update 4 Has Arrived!

Changes made:

  • Fight4Life now uses the 1.8 protocol meaning you can log in using any client version from 1.8 to 1.12.2!
  • Combat mechanics in the Factions realm have been modified to make shields/blocking more useful
  • Shields can now be decorated in factions!
  • You can now dual wield items in factions!
  • A few optimizations were made in the server configurations
