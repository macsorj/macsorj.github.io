What does Minecraft 1.13 mean for Fight4Life?

July 27, 2018 macsorj Uncategorized 0

Hello everyone, so you may know that on July 18th, the Minecraft 1.13 Aquatic Update was officially released. This is the single largest update in Minecraft history, although it may not look like it. For most of you this update probably doesn’t seem very significant because only a few updates to the gameplay and features were made. However, the reason I say this update is the biggest one yet is because of the massive backend API changes that the game has undergone. The primary focus of 1.13 for the game developers was all the technical stuff, not the new content.

So what does this mean for Fight4Life? Because of the huge changes that are happening, it will likely be many months before we can support Minecraft 1.13 and the updates that will follow. This is because the SpigotMC team and all of the plugin developers have tons of work to do when they update their stuff. I do not think we will be able to continue running the 1.8 API because of all the changes that must be made, so we might have to drop support for 1.12 and all versions that come before it when we finally move to 1.13. A few plugins that the server runs right now may have to be removed in the future.

This update will not significantly impact gameplay on Fight4lifeMC, but will seriously slow down progress towards new content and features on the server. Right now there’s not much to do except wait, because like I said before, the SpigotMC team has a lot of work ahead of them before they are able to release a stable build of Spigot. It will likely be anywhere from 2 to 9 months before we can begin the update process. If you want to learn more about the changes to Spigot and the new API, you can read this update written by md_5, a SpigotMC administrator and developer: https://goo.gl/1WX9su

One last thing: Yeah I know the server is kinda dead right now lol but i’m going to work on it soon. I’ve been really busy with life ‘n’ stuff.
