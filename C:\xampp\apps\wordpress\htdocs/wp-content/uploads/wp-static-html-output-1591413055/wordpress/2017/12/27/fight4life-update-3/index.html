Fight4life Update 3

December 27, 2017 macsorj Uncategorized 1

After nearly a month of development, Fight4Life Update 3 has finally been completed!

Changes made:

  • Skyblock has been moved from Survival to its own realm
  • A new spawn now serves as a hub to connect to all of the realms. The old spawn is now the spawn for the survival realm
  • The old Minecraft combat system is now in use in all realms except for Skyblock
  • The factions realm was introduced
  • Several broken plugins were removed
  • The vote sites were updated
  • Crate rewards in the factions realm were updated
  • The server has been upgraded to 24GB of RAM from 12
  • All server files were moved to solid state storage, vastly improving the loading times
